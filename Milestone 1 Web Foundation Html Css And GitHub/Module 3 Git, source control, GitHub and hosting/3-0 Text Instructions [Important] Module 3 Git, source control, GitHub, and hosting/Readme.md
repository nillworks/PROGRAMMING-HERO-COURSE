আজকের মডিউল এ গিট নিয়ে কথা বলা হয়েছে। তোমাদের অনেকেরই জন্য গিট জিনিসটা একদমই
নতুন। এবং অনেক ডিফারেন্ট একটা জিনিস। তবে চিন্তা করার কিছু নাই। কয়েকদিন করতে
করতে দেখবা জিনিসটা পানির মতো সোজা হয়ে গেছে। আজকের প্রথম ছয়টা ভিডিও বুঝে নেয়ার
চেষ্টা করবে। কোন কারণে কোন একটা ভিডিওতে আটকে গেলে। সেই ভিডিও পুরাটা দেখো। এই
প্রথম ছয়টা ভিডিও এর কাজ এখনই ক্লিয়ার করে রাখতে হবে।

কারো মাথার উপর দিয়ে যাবে। কেউ কেউ ধরতে পারবে আর বেশিরভাগ পেঁচগি লাগায় ফেলবে।
