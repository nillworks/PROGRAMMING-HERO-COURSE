# Git & GitHub Quiz Answers

This README contains the questions and answers for quick practice.

---

## Question 01

**What is Git?**

**Options:**

- A. A web-based hosting service
- B. A package manager
- C. A build tool
- D. A distributed version control system

**Answer:**  
D. A distributed version control system

---

## Question 02

**What is GitHub?**

**Options:**

- A. A cloud-based service for hosting Git repositories
- B. A local Git client
- C. A cloud terminal
- D. A CSS framework

**Answer:**  
A. A cloud-based service for hosting Git repositories

---

## Question 03

**Which command initializes a new Git repository?**

**Options:**

- A. git clone
- B. git init
- C. git start
- D. git create

**Answer:**  
B. git init

---

## Question 04

**Which command stages files for commit?**

**Options:**

- A. git commit
- B. git add
- C. git push
- D. git stage

**Answer:**  
B. git add

---

## Question 05

**Which command saves a snapshot in the local repo?**

**Options:**

- A. git save
- B. git commit
- C. git push
- D. git record

**Answer:**  
B. git commit

---

## Question 06

**Which command pushes commits to the remote repo?**

**Options:**

- A. git pull
- B. git add
- C. git push
- D. git deploy

**Answer:**  
C. git push

---

## Question 07

**What is a “remote” in Git?**

**Options:**

- A. A local branch
- B. A reference to a remote repository
- C. A staging area
- D. A file backup

**Answer:**  
B. A reference to a remote repository

---

## Question 08

**How do you create and switch to a new branch in one step?**

**Options:**

- A. git branch new
- B. git checkout -b new
- C. git push new
- D. git new branch

**Answer:**  
B. git checkout -b new

---

## Question 09

**What does git merge do?**

**Options:**

- A. Reverts a commit
- B. Combines branches into the current branch
- C. Deletes a branch
- D. Fetches data from remote

**Answer:**  
B. Combines branches into the current branch

---

## Question 10

**What is a pull request?**

**Options:**

- A. A local Git command
- B. A file deletion
- C. A request to merge changes via GitHub
- D. An ignore list

**Answer:**  
C. A request to merge changes via GitHub
